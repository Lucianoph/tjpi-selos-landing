Instalação
Como Instalar

Siga os passos abaixo para instalar e configurar o aplicativo pela primeira vez.

01
Baixe o instalador
Clique em Download — Instalador (.exe) na seção acima. O arquivo TJPI_Selos_Setup.exe será salvo na sua pasta de Downloads.
02
Libere o arquivo no Windows Defender
O Windows pode exibir um alerta de segurança pois o aplicativo ainda não possui assinatura digital. Para prosseguir:

→ Clique em "Mais informações" na tela azul do SmartScreen
→ Clique em "Executar assim mesmo"
Isso é normal para aplicativos sem certificado digital. O software é seguro.
03
Execute o instalador
Clique duas vezes no arquivo baixado e siga as instruções na tela. O instalador irá:

✓ Criar atalho na Área de Trabalho
✓ Criar entrada no Menu Iniciar
✓ Copiar os arquivos necessários

Não é necessário ter Python ou qualquer outra dependência instalada.
04
Aguarde o servidor conectar
Na primeira vez que abrir, o aplicativo irá se conectar ao servidor. Se o servidor estiver em modo de espera (Render Free), pode levar até 1 minuto para conectar. Uma barra de progresso será exibida.
Não feche o aplicativo durante a conexão inicial. O servidor acorda automaticamente.
05
Crie sua conta
Após conectar, vá na aba "Criar Conta" e preencha:

• Nome completo
• Login (sem espaços)
• Senha (mínimo 4 caracteres)

Cada computador permite apenas um cadastro gratuito de 30 dias. O sistema identifica sua máquina automaticamente.
Cada máquina = 1 conta = 30 dias grátis
06
Pronto! Configure e use
Após o login, o aplicativo principal será aberto. Tem a opção de login manual no site do TJPI e para login automático Configure o Login/cartório e comece a usar. Seu login fica salvo — nas próximas vezes, o aplicativo entra automaticamente.

CallMeBot — Alertas no WhatsApp em Tempo Real
O sistema usa o serviço CallMeBot para enviar alertas direto no seu WhatsApp quando um novo período de restituição for detectado. A ativação é gratuita e leva menos de 2 minutos — basta seguir os passos abaixo.
1
Adicione o número do CallMeBot nos seus contatos
Antes de enviar qualquer mensagem, salve o número abaixo na sua agenda com o nome "CallMeBot" (ou qualquer nome que preferir). Isso é necessário para o WhatsApp permitir a conversa.
+34 684 73 40 44
Este é o número oficial do CallMeBot para a API do WhatsApp.
2
Envie a mensagem de ativação para o CallMeBot
Abra o WhatsApp, inicie uma conversa com o contato CallMeBot que você acabou de salvar e envie exatamente a mensagem abaixo (em inglês, como está):
I allow callmebot to send me messages
Simulação da conversa
I allow callmebot to send me messages
✓✓
CallMeBot
API Activated for your phone number.
Your APIKEY is 123456
Thanks for using the service
agora
O bot pode demorar alguns minutos para responder na primeira vez. Se não responder em 5 minutos, tente enviar a mensagem novamente."Talvez ele esteja sobrecarregado e irá te enviar um novo número do CallMeBot para você utilizar!"
3
Copie o seu APIKEY (Token)
O CallMeBot irá responder com uma mensagem contendo o seu APIKEY — esse é o seu token pessoal. Copie apenas os números, como no exemplo:
123456
O APIKEY é uma sequência numérica única gerada para o seu número. Guarde-o para usar no próximo passo.
4
Insira o Token e o Número no Aplicativo
Abra o aplicativo TJPI Selos, vá em Configurações → Alertas WhatsApp e preencha os dois campos:
Campo: Token do Bot (APIKEY)
123456
Cole aqui o APIKEY recebido do CallMeBot
Campo: Número WhatsApp
5586999990000
Seu número com código do país: 55 (Brasil) + DDD + número, sem espaços ou traços
5
Salve e Teste a Integração
Clique em "Salvar" ou "Ativar Alertas". O aplicativo enviará uma mensagem de teste via CallMeBot para confirmar que tudo está funcionando.
✅ Se você receber uma mensagem de teste no WhatsApp, a integração está ativa e pronta para uso.
6
Pronto! Aguarde os Alertas Automáticos
Sempre que o Monitor detectar um novo período de restituição, você receberá uma mensagem como esta no WhatsApp:
Exemplo de alerta recebido

💰 Valor: R$ 1.250,00
🏦 Banco / Agência: 001 / 0042
⏱ Detectado em: 14:32:07

O Monitor Automático precisa estar ativo para os alertas serem enviados. Certifique-se de que o app está aberto e o monitor iniciado na aba Monitor.
O Windows bloqueou o arquivo, o que faço? ▼
Isso ocorre porque o aplicativo não possui assinatura digital. É um comportamento padrão do Windows SmartScreen. Na tela azul de alerta, clique em "Mais informações" e depois em "Executar assim mesmo". O arquivo é seguro.
O aplicativo demorou muito para abrir na primeira vez. É normal? ▼
Sim. O servidor usa o plano gratuito do Render, que "dorme" após inatividade. Na primeira abertura do dia pode levar até 1 minuto para o servidor acordar. Uma barra de progresso é exibida durante esse tempo. Nas tentativas seguintes (mesmo dia) é instantâneo.
Posso instalar em mais de um computador com a mesma conta? ▼
Não. Cada conta fica vinculada ao hardware do computador onde foi criada. Cada máquina permite apenas um cadastro gratuito de 30 dias. Se você trocar de computador, entre em contato com o administrador para desvincular sua conta do hardware anterior.
Meu período de teste expirou. Como renovar? ▼
Entre em contato com o administrador do sistema para renovar sua licença. A renovação é feita diretamente pelo painel administrative, adicionando mais dias à sua conta sem perda de dados ou configurações.
O monitor parou de funcionar do nada. O que pode ser? ▼
As causas mais comuns são: (1) computador entrou em modo de suspensão — reinicie o monitor após acordar a máquina; (2) sua sessão expirou após 8h — faça login novamente; (3) o portal do TJPI pode estar em manutenção temporária. Verifique os logs na aba de status do aplicativo.
Como atualizar para uma nova versão? ▼
Baixe o novo instalador nesta página e execute normalmente. O instalador sobrescreve a versão anterior mantendo suas configurações. Não é necessário desinstalar antes.
